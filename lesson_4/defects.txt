В файле string_utils.py есть дефекты:
1. Опечатка в названии метода:
Метод capitilize должен быть написан как capitalize.
2. Метод trim:
В Python строковый метод str.removeprefix() уже удаляет пробелы в начале строки, поэтому цикл while не нужен.
3. Метод contains:
Использование метода string.index(symbol) > -1 для проверки вхождения подстроки неэффективно. Система не корректно работает со строкой, которая не содержит ни одного символа.
4. Метод delete_ symbol:
Проверка наличия символа перед его удалением избыточна.
5.Описание методов:
В описаниях методов, где указаны примеры, лучше использовать корректные форматы вызовов.
6. Метод end_with:
Метод end_with должен быть написан как ends_with для соответствия общепр
